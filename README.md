# Begin your path to Poshmark ambassador! (WIP)
This is a tool that automates listing sharing for your Poshmark marketplace account.

Currently, the tool is only able to share listings in your following list.

This is still very much a work in progress. Future functions will include:
- designating list sharing from following or followers list
- self share items in your closet
- sharing to relevant parties based on tag and item name
- a lot more error handling

This is my first open-source project, if there are any suggestions or issues, let me know!
